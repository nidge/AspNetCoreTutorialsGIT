This is the complete finished Create a web app with ASP.NET Core MVC found here - https://docs.microsoft.com/en-us/aspnet/core/tutorials/first-mvc-app/start-mvc?view=aspnetcore-3.0&tabs=visual-studio

To create the database before running this app, you will probably need to go to Package Manager Console and run 
Update-Database

You should then be able to view this database by going to View --> SQL Server Object Explorer.  You may need to connect to the Local DB - the name of the server can be seen in appsettings.json
